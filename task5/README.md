Интерполяция - использование переменных в текстовой строке.

подстановка значений в текстовую строку является интерполяцией.

Внутри одной переменной нельзя проводить интерполяцию из других двух и более переменных
err:  bad example

Именно локальныезначения переменных, записанных в locals.tf позволяют что-то вычислять при работе терраформа.

Заведены переменные, обрамлены кавычками.
введена доп переменная test, чтобы посмотреть из terraform.console
